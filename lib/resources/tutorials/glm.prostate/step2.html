The GLM algorithm uses a training dataset to build a model. In this tutorial, we will use the popular Prostate cancer dataset for model building.

The prostate dataset is provided here. Please, download it and save it on your disk.

To upload the dataset into H2O application, please, use the menu option Data > Upload or direct link.

Alternatively, Data can be imported from a URL directly into H2O application, please, use the menu option Data > Import Files or direct link.
