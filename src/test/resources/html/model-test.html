WebDriver Extension

Simplify testing with WebDriver Extension!

WebDriver Extension is a framework on top of WebDriver/Selenium 2 designed to make testing easier and more fun by puting focus on test readability, reusability and maintainability.

Readability

By providing static methods for commands and asserts, a.k.a Bot Style Testing, the tests gets readable as human instructions:

E.g. Login Test

open(loginPage);
type("andi", loginPage.username);
type("mypassword", loginPage.password);
click(loginPage.login);
assertPage(welcomePage);

Reusability

By modeling pages as objects, a.k.a Page Object Pattern, the css-selectors or prefered locator only needs to be defined once

Maintanability

If the html for a page would change for some reason there is no need to change the locators on every place it is used, just change the locator in the Object where it is defined.
