Data: Export Files
====================

Data files can be exported as S3, HDFS or NFS. 

**Src key** 

  Enter the key associated with the data. 


**Path**

  Enter the file path to S3, HDFS, NFS, or URL for the location of the data. 

**Force** 

  To overwrite existing files, check this checkbox.
